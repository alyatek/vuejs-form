## Installation

### NPM

```bash
npm install --save-dev vuejs-form
```

### Yarn

```bash
yarn add vuejs-form --save
```

### CDN

> **Minimized CDN**
```bash
<script src='https://unpkg.com/vuejs-form@1.1.0/build/vuejs-form.min.js'></script>
```

> **Non Minimized CDN**
```bash
<script src='https://unpkg.com/vuejs-form@1.1.0/build/vuejs-form.js'></script>
```
